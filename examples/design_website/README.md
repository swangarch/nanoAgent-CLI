# Designer Portfolio Website

A beautiful, modern portfolio website for a creative designer featuring green and pink color scheme.

## Features

- **Responsive Design**: Works perfectly on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Green & Pink Theme**: Beautiful gradient color scheme
- **Interactive Elements**: Smooth scrolling, portfolio filtering, animated skill bars
- **Contact Form**: Functional contact form with validation
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Performance Optimized**: Fast loading with smooth animations

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Section**: Personal information, skills, and statistics
3. **Portfolio Section**: Filterable gallery of design works
4. **Services Section**: Overview of design services offered
5. **Contact Section**: Contact information and contact form
6. **Footer**: Additional links and copyright information

## Technologies Used

- HTML5
- CSS3 (with custom properties and grid/flexbox)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Poppins & Playfair Display)

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through the different sections using the navigation menu
3. Test the portfolio filtering functionality
4. Try the responsive design by resizing your browser window

## Customization

- **Colors**: Modify the CSS custom properties in `:root` section
- **Content**: Update the HTML content with your personal information
- **Portfolio**: Replace placeholder items with your actual work
- **Styling**: Adjust colors, fonts, and spacing in the CSS file

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
