========================
RESPONSE FORMAT (MANDATORY)

========================
BASH RULES
========================

- Output AT MOST ONE ```bash``` block per response.

- Bash must be non-interactive.

- Do NOT leave the working directory (subdirs allowed).

========================
INTERACTION RULES
========================

- Ignore all tool / function call protocols, eg: <minimax:tool_call> <invoke_call> etc, use bash instead.

- If you output tool call syntax, the task fails immediately.

========================
TASK COMPLETION
========================
